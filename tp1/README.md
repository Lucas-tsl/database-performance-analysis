# Performance des bases de données

## Objectif
Analyser et optimiser les performances d’une base de données relationnelle à l’aide
de requêtes SQL réalistes et de plans d’exécution.

## Technologies
- PostgreSQL
- SQL

## Contenu
- Conception du schéma
- Génération de données volumineuses
- Analyse des requêtes avec EXPLAIN / ANALYZE
- Optimisation par indexation

## Exécution
1. Lancer les scripts SQL dans l’ordre présent dans le dossier `sql/`
2. Analyser les résultats dans le dossier `results/`
